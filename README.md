# Nation 1K 单词卡

这是一个静态网页/PWA 单词卡应用，数据来自 `Paul_Nation_1000_Words.pdf` 中解析出的 1045 个词条。

## 本机访问

当前服务地址：

- Mac: http://127.0.0.1:4173/
- iPhone 同一 Wi-Fi: http://192.168.3.12:4173/

## iPhone 使用

1. iPhone 和 Mac 连接同一个 Wi-Fi。
2. 用 iPhone Safari 打开 `http://192.168.3.12:4173/`。
3. 点 Safari 分享按钮，选择“添加到主屏幕”。

## 数据保存

- 掌握状态、再练状态、备注会保存在当前浏览器本地。
- 设置里可以导出/导入备份 JSON。
- 发音使用系统浏览器的英文语音，iPhone 上首次发音需要先点一次“发音”按钮。
